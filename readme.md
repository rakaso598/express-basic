## express 서버 기초 연습

### 연습하려는 것

- Router (맵핑)
- 동적 라우팅
- Middleware
- 미들웨어 에러처리
- 정적 파일 배포

### 경로 목록

- localhost:3010/ | 이곳은 인덱스입니다
- localhost:3010/board | 이곳은 board 경로입니다
- localhost:3010/board/:boardId | ID가 boardId인 게시판에 오셨습니다.
- localhost:3010/user | 이곳은 user 경로입니다
- localhost:3010/user/:userId | ID가 userId인 게시판에 오셨습니다.
